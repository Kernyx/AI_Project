# Загруженные фотографии

В этой папке хранятся оригинальные загруженные изображения татуировок.

Демо-фотографии из этой папки можно коммитить, если нужно показывать проект
с уже подготовленными примерами. Реальные персональные данные сюда лучше не
добавлять без согласия людей на публикацию в репозитории.
